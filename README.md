# 🍱 배재고 점심 급식 알리미

배재고등학교 학생들을 위한 **실시간 점심 급식 알림 웹 서비스**입니다.  
나이스(NEIS) API를 활용해 매번 번거롭게 사이트 들어갈 필요 없이, 지금 점심 메뉴가 뭔지 0.1초 만에 확인합니다.



## 🚀 주요 기능
- **자동 식단 추적**: 오늘 메뉴가 없으면 자동으로 다음 등교일 메뉴를 찾아줍니다.
- **점심(중식) 특화**: 조식, 석식 정보는 필터링하고 오직 우리가 먹는 **점심 메뉴**를 먼저 보여줍니다. 그러나 기숙사생을 위해 조식과 석식 정보도 표시됩니다.
- **자동 날짜 로직**: 금요일 저녁/주말에 접속해도 당황하지 않고 월요일 메뉴를 띄워줍니다.

## 🛠️ 개발 정보 (기술 스택)
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: [나이스(NEIS) 교육정보 개방 포털 API](https://open.neis.go.kr/)
